What I have learned about the Web

The Web is a collection of HTML documents that includes hyperlinks, or links for short. One can view HTML documents using a browser such as Firefox, Chrome, or my new preference Safari. Browsers connect to the internet to acquire files that are hosted on computers known as servers. All of this is done by using HTTP; a protocol that allows for the communication between a browser and server.


What I have learned about HTML (Hyper Text Markup Language)
HTML is straightforward

Tags are used to format HTML Elements

An Element is contained by an opening tag and a closing tag.

Anchor Tags are utilized to include links to other websites.
Please click on the following link to learn more about anchor tags. Anchor Tags Tutorial

White Space is controled by including inline or block elements.

The break is an example of an inline tag and the paragraph tag is an example of a block tag.
